# IT-Support-E-commerce

TechDesk Pro is a Next.js application for IT support and ecommerce-focused service workflows.

## Repository layout

- `techdesk-pro/` — main application source code.
- `techdesk-pro/supabase/migrations/` — database migration SQL files.

## Run locally

```bash
cd techdesk-pro
npm install
npm run dev
```

The app starts on the default Next.js development port (`http://localhost:3000`).
